# Docs

本目录存放仓库的说明性文档，用于帮助开发成员和 AI Agent 快速理解框架结构、使用方式和工作流。

---

## 目录内容

### `methodology.md`
说明这套框架的方法论、设计原则和适用场景。

### `quick-start.md`
快速开始文档，适合第一次接触仓库的人阅读。

### `command-reference.md`
命令参考文档，列出各类指令、触发词或工作流入口。

### `repository-index.md`
仓库索引文档，说明整个仓库的结构、职责分层、加载顺序和执行闭环。

### `README.md`
当前文件，作为 `docs/` 目录的入口说明页。

---

## 推荐阅读顺序

如果你是第一次接触这套框架，建议按以下顺序阅读：

1. `../README.md`
2. `../AGENTS_CORE.md`
3. `repository-index.md`
4. `quick-start.md`
5. `methodology.md`
6. `command-reference.md`

---

## 使用建议

- 想快速上手：先看 `quick-start.md`
- 想理解整体结构：先看 `repository-index.md`
- 想理解方法论：看 `methodology.md`
- 想查命令：看 `command-reference.md`

---

## 目录原则

`docs/` 下的文档应尽量保持：

- 面向阅读
- 结构清晰
- 不重复写大段内容
- 入口文档简短，详细说明放到专门文件中
