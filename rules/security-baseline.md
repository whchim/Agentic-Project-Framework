# 安全基线

## 输入安全
- 所有外部输入都要校验
- 所有参数都要考虑边界值
- 所有文件上传都要验证类型和大小

## 数据安全
- 敏感数据必须加密
- 日志中禁止输出密钥、密码、token、身份证号等敏感信息
- 数据库存取使用参数化查询

## 鉴权与授权
- 受保护接口必须鉴权
- 权限边界要明确
- 最小权限原则

## Web 安全
- 防 XSS
- 防 CSRF（适用场景）
- 防 SQL 注入
- 限流保护敏感接口

## 依赖安全
- 不引入明显弃用或高风险包
- 发现漏洞要评估修复
- 新依赖要说明理由

## AI 场景补充
- 禁止把敏感信息直接发给外部模型
- Prompt 和输出要做必要保护
- 重要生成结果必须验证
