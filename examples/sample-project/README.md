# Sample Project

这是一个演示如何使用 Agentic Project Framework 的示例项目。

## 场景
做一个“用户登录”功能，演示从需求到归档的完整流程。

## 包含内容
- Product Spec
- Design Doc
- Exec Plan
- Validation Report
- Archive Summary

## 使用方式
1. 先读 Product Spec
2. 再读 Design Doc
3. 再看 Exec Plan
4. 最后看验证和归档

## 目的
帮助新手理解：
- 需求怎么写
- 设计怎么写
- 计划怎么拆
- 怎么验证
- 怎么归档
